# Architecture Overview

## Layers
1. **Data Ingestion Layer:** EHR, sensors, imaging; privacy-compliant
2. **Cloud Infrastructure:** Dockerized services, scalable servers
3. **Backend (API & AI):** FastAPI, generative models, twin logic
4. **Frontend (Dashboard):** React app for visualization, analytics
5. **Testing & DevOps:** Automated unit/integration tests, CI/CD ready

## Technology Choices
- Cloud: AWS/Azure/GCP
- API: FastAPI (Python)
- ML: PyTorch/TensorFlow/Hugging Face
- UI: React + MUI
- DB: PostgreSQL/MongoDB
- Infra: Docker, Terraform
